# TrafficLights (신호등 플러그인)

## 사용처
#### 마인크래프트 야생 플레이시 사용할 수 있습니다.

## 사용방법
#### [/tl] 명령어를 사용하여 비/활성화 할 수 있습니다.
#### 1분마다 50%의 확률로 빨간 신호등이 활성화됩니다!
#### 빨간 신호등인 상태에서 움직이면 플레이어는 사망합니다.

## 주의사항
#### 💡 채팅 도배 방지를 위해 시스템이 활성화되어있는 경우 다잉메시지가 출력되지 않습니다. (플레이어가 명령어를 사용한 월드만 해당)
